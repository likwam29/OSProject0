In order to run this program you will first have to extract all the files and navigate to the directory in your terminal.
Once you are in the directory type "make"
Once the project has compiled you can run the program by typing in "./program Data.bin csvFile".
The ./program is not changable, but the Data.bin can be named anything as long as it has the file extension.bin and lives in the programs directory.
The csvFile is what you want your output csv file to be named.
